> This folder contains three analytical projects completed as part of postgraduate study in Business Analytics at UNSW Sydney. They demonstrate applied machine learning, data preparation, and model evaluation skills that underpin the AI and analytics work across this portfolio.

---

## Projects

### 01 — Best Seller Classification (Amazon)
**File:** `Best seller classification Amazon.ipynb`

Classifies Amazon product listings as best sellers or non-best sellers using product metadata and review features. Explores feature engineering on text and numerical attributes, compares multiple classification algorithms, and evaluates model performance using accuracy, precision, recall, and F1 score.

**Key skills demonstrated:**
- Feature engineering from structured and semi-structured product data
- Binary classification: Logistic Regression, Random Forest, Gradient Boosting
- Model comparison and selection with cross-validation
- Handling class imbalance in real-world datasets

---

### 02 — Heart Disease Prediction Model
**File:** `Heart_Disease_Prediction_Model.ipynb`

Builds a predictive model for heart disease risk using clinical features from the UCI Heart Disease dataset. Covers exploratory data analysis, feature selection, model training, and ROC-AUC evaluation. Demonstrates application of ML in a healthcare context, directly relevant to aged care technology work.

**Key skills demonstrated:**
- Exploratory data analysis and visualisation with matplotlib and seaborn
- Feature selection and correlation analysis
- Binary classification with healthcare implications
- ROC curve analysis and threshold optimisation
- Responsible ML: understanding false negative costs in clinical contexts

---

### 03 — Smartphone Classification
**File:** `Smartphone Classification.ipynb`

Classifies smartphones into price range categories based on technical specifications. Uses a multi-class classification approach and demonstrates feature importance analysis to identify which hardware attributes most influence pricing tier.

**Key skills demonstrated:**
- Multi-class classification with scikit-learn
- Feature importance ranking and interpretation
- Data normalisation for numerical feature sets
- Business interpretation of model outputs
